# HostMaps

A tool to generate custom maps to local attractions as a host. Select your address, pick nearby attractions, and download a PNG with routes and travel times. Uses Google Maps Platform (Places, Directions, Static Maps)

See it here: [hostmaps.krets.com](https://hostmaps.krets.com)

## Features
- **Smart Search**: Auto-finds nearby attractions (ski resorts, concert venues, etc.) or manually add your own.
- **Route Calculation**: Shows drive times and draws routes on the map.
- **Custom Export**: Downloads a clean PNG with a legend, perfect for printing or digital guidebooks.
- **Mobile Friendly**: Works on your phone so you can build maps on the go.
